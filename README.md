# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Far-200/portfolio-site.git
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

---

# 🌍 Deployment

The website is deployed using **Vercel** for fast global delivery.

Every push to the `main` branch automatically triggers a deployment.
